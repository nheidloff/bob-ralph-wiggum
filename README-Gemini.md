# Gemini

Init Gemini:

```
git clone https://github.com/nheidloff/bob-ralph-wiggum.git
cd bob-ralph-wiggum

export GEMINI_API_KEY="xxx"
export GIT_USER_EMAIL="xxx"
export GIT_USER_NAME="xxx"
sh init-gemini.sh
```

Start Container: 

```
sh start-container-gemini.sh
```

Run Gemini:

```
sh ../ralph/iteration.sh
# or
sh ../ralph/loop.sh
```
